# notebooklm-habitos-saudaveis-jovens
Projeto desenvolvido utilizando o NotebookLM para estudar hábitos saudáveis voltados ao público jovem de 15 a 25 anos. O trabalho reúne curadoria de fontes confiáveis, testes de prompts, análise dos resultados gerados pela IA e um miniguia de estudos contendo resumos, glossário e prompts reutilizáveis para futuras consultas.
